# Alize Project

## Generating your token

1. Go to https://artifacts.jll.com/ui/user_profile
2. Click Generate identity token
3. Add any description, click next
4. Copy that token for the below steps

## Getting This Project

This project is shared via OneDrive. To access it:

1. **If you received a OneDrive link**: Click the link and download/sync the folder
2. **If you have direct access**: The folder is in OneDrive at `Documents/alizeproject`

## Setup After Downloading

1. Copy `.npmrc.example` to `.npmrc` and add your JLL auth token, then on the keyboard use Cmd + S to save
2. Run `npm install` to install dependencies
3. Run `npm run dev` to start the development server

That's it!
